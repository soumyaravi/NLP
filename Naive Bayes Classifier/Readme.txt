File descriptions -

1. nblearn.py - The program for task 1 for training the model.
2. nbclassify.py - The program for task 1 for classifying the files.
3. nblearn10.py - The program for task 2 for training the model with only 10% of the training data. nbclassify.py is used for classifying the model.
4. nbclassify.py - The program for task 2 for classifying the files.
5. nblearn3.py - The program for task 3 for training the model. 
6. nbclassify3.py - The program for task 3 for classifying the files.
7. nbcheck.py - The program that calculates the precision, recall, f1 score and weighted average.
8. nbmodel.txt - The output for task 1 of training model.
9. nboutput.txt - The output for task 1 after classifying the files.
8. nbmodel2.txt - The output for task 2 of training model.
9. nboutput2.txt - The output for task 2 after classifying the files.
8. nbmodel3.txt - The output for task 3 of training model.
9. nboutput3.txt - The output for task 3 after classifying the files.
